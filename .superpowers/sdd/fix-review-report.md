### Fix 4: X-User-ID spoofing ✅
**File:** `app/user/internal/service/social.go`
- Added HMAC-SHA256 verification using `crypto/hmac` + `crypto/sha256`.
- If `X-User-ID-Signature` header is present:
  - Reads `HMAC_SECRET` env var.
  - Computes `base64(HMAC-SHA256(secret, X-User-ID))` and compares with the provided signature.
  - On mismatch, returns `codes.Unauthenticated`.
- If `X-User-ID-Signature` is absent:
  - Falls back to trusting `X-User-ID` directly (dev mode behind Envoy).
- Build: ✅ `go build ./cmd/user/`
